# Shopping List App

A Flutter application to manage your grocery list, integrated with Firebase Realtime Database.

## Setup

1. **Firebase**:
   - Create a Firebase project at [Firebase Console](https://console.firebase.google.com/).
   - Enable **Realtime Database**.
   - Update `lib/constants.dart` with your database URL.

2. **Run the app**:
   ```bash
   flutter run
   ```

## Features
- Add new items with quantity and category.
- Remote list synchronization.
- Swipe-to-delete with backend integration.
- Modern dark theme.
